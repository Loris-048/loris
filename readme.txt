========================================================================
             Loris 绘图工作平台 - 源码拆分与安全局域网共享方案
========================================================================

欢迎使用全新重构的 Loris 绘图工作平台！
我们通过“物理隔离”的方式实现了高安全、防拷贝、防泄密的标准企业级开发与分发架构。

------------------------------------------------------------------------
一、 目录结构说明
------------------------------------------------------------------------
Loirs/
├── public/                 --> 前端展示源码（开发阶段，干净清晰，可精确修改）
│   ├── index.html          --> 纯净的 HTML 骨架（不含任何 JS 与 CSS）
│   ├── styles.css          --> 所有的界面样式（~72KB）
│   └── app.js              --> 前端核心交互逻辑（~560KB，已集成安全代理拦截）
├── config.json             --> 🔑 敏感密钥与端口配置文件（放在后端，同事无法读取）
├── server.js               --> 🚀 局域网/本地 Node.js 核心中转服务器
├── build.js                --> 📦 简易压缩打包编译工具（一键把 public 代码合成为单文件）
├── run.bat                 --> 双击一键启动服务器并打开网页
└── readme.txt              --> 本说明文件

------------------------------------------------------------------------
二、 🔒 它是如何做到“防盗、防复制代码和密钥”的？
------------------------------------------------------------------------
1. API 密钥防盗（防看、防刷）：
   - 只要在 `config.json` 中填入了 `banana_token` 或 `modelscope_token`，
     启动服务后，前端页面的“秘钥输入框”会自动锁定并显示为 `********`，
     同事无法在浏览器里看到或更改您的密钥。
   - 所有向 OpenAI, Gemini 或魔塔的生图、聊天请求全部通过您电脑本机的 `server.js` 
     在后台默默附带上真实的 Key 发送。
   - 同事通过 F12 只能看到 `/api/proxy` 这个本地局域网地址，绝对接触不到真实的 API 密钥。

2. 核心代码防拷贝：
   - 所有的生图任务规划逻辑、意图分类算法都集成在 `app.js` 中。
   - 如果您要将网页分发给同事：
     * 不要发送整个文件夹或 `.html` 源码。
     * 只需在本机双击 `run.bat` 启动服务，然后将启动日志里输出的局域网地址 
       （例如 http://192.168.1.100:3000）分享给同事。
     * 同事通过这个网址可以直接完美使用所有功能，但是他们的电脑上【没有任何源码文件】，
       一旦您在自己的电脑上关掉 `run.bat` 窗口，他们就无法再次访问和工作。

------------------------------------------------------------------------
三、 ⚙️ 使用说明（极简两步）
------------------------------------------------------------------------
第一步：配置 API 密钥
1. 打开 `config.json` 文件。
2. 填写您的密钥和代理地址，如：
   {
       "port": 3000,
       "banana_token": "您的Banana_API_Token_秘钥",
       "modelscope_token": "您的魔塔_API_Token_秘钥",
       "chat_api_base": ""
   }
3. 保存并关闭文件。

第二步：一键启动
1. 双击运行 `run.bat` 文件。
2. 电脑会自动开启黑色的命令行窗口（本地服务器），并自动在您的默认浏览器打开 
   http://localhost:3000，即可立即开玩！
3. 分享给同事：将命令行窗口中显示的 `http://192.168.x.x:3000` 网址复制发给同一个局域网内的同事，他们就能在各自电脑上一起使用了。

------------------------------------------------------------------------
四、 📦 如果需要“双击即用”的独立单文件怎么办？
------------------------------------------------------------------------
如果您需要带一份完美的独立 HTML 单文件回家使用，或者作为轻量级版本备用：

1. 在当前目录下，打开终端，运行命令：
   node build.js

2. 脚本会在瞬间编译合并代码，并在同目录下自动创建一个 `dist/` 文件夹。
3. 里面的 `dist/index_bundled.html` 就是生成的完整独立单网页！
   * 这个打包版本已安全过滤并删除了所有开发者的内部中文注释和调试语句，极大增加了反编译阅读门槛。
   * 特性自适应：如果通过 `file://` 双击打开它（没有启动后端），它会自动判定并【安全回退】为直连官方 API 的老模式（可以在前端手动输入 API 密钥继续玩），完美保持了单文件的超强便携性！

------------------------------------------------------------------------
五、 🤝 开发维护建议
------------------------------------------------------------------------
1. 以后若需要对平台进行任何优化、修改或增加功能，一律直接在 `public/app.js` 或 
   `public/styles.css` 中精细修改即可，再也不用面对那个 80 万字节的臃肿万行大文件了！
2. 修改完确认无误后，运行 `node build.js` 即可生成最新的单文件版本。

------------------------------------------------------------------------
六、 🚀 GitHub Pages 移动端极速云同步 (一箱装走随身包方案)
------------------------------------------------------------------------
为了让您在手机端也能无缝、全自动同步使用最新开发的生图排版与样式，
我们将整个 Git 时光机和上传工具统统打包收纳进了 Loirs 这个“单一文件夹”里：

1. 📂 随身包中的新角色：
   - Loirs/index.html   --> 专门用于 GitHub Pages 在线端的生产首页（由 build.js 自动生成）
   - Loirs/deploy.bat   --> 双击一键打包编译并推送更新到您的 GitHub 仓库
   - Loirs/.gitignore   --> 安全防火墙，100% 自动拦截本地 config.json 与 dist/ 离线包

2. 📱 如何同步到手机浏览器？
   第一步（仅需做一次）：
     - 双击运行 Loirs/deploy.bat。
     - 当黑窗口提示输入您的 GitHub 仓库地址时，粘贴您的 HTTPS 链接
       （例如：https://github.com/您的用户名/您的仓库.git）并回车。
     - 脚本会自动记住这个地址并进行首次推送。
   
   第二步（在 GitHub 上开启在线站点，仅需做一次）：
     - 打开您的 GitHub 仓库网页 ➔ Settings (项目设置) ➔ 找到左侧 Pages。
     - 在 Branch 栏，将分支切换为 main，点击右侧的 Save (保存)。
     - GitHub 就会自动为您生成一个永久在线网址（如：https://您的用户名.github.io/loris/）。

3. 🔄 以后日常修改后如何更新：
   - 您在 Loirs/public 里的开发源码做任何功能修改。
   - 只要双击运行 Loirs/deploy.bat。
   - 窗口闪烁 3 秒后全自动关闭。
   - 拿起手机刷新网页，最新优化功能立刻就上线活了！

4. 💼 U 盘备份/换电脑随身带走说明：
   - 以后换电脑，您【只需要打包带走 Loirs 这一个文件夹】！
   - 里面不仅携带了所有的时光机历史、一键部署脚本，还包含前后台核心源码。
   - 换电脑后，本地 run.bat 依然双击即用，deploy.bat 依然能够完美一键上传更新，实现终极的极客便携性！
